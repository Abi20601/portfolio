Disk Space Manager

Project information

  • Category: Desktop Application
  • Tech Stack: Python, Tkinter, C++
  • Project date: July 2023
  • Project URL: github.com/coderSuren/Codebrewers-hackathon
Project Description:

This project was developed during the Codebrewers Hackathon conducted by Tally solutions under the theme named "Wizard of System Programming". This is a Python application built using the Tkinter library and provides a user-friendly interface to scan for duplicate files and manage system memory efficiently. With this app, you can quickly locate and remove duplicate files, freeing up infrequently accessed files, and viewing disk memory utilization based on drive, directory, file types, etc.
