Project Description:
This project involves developing a web application that streamlines the process of attendance-taking by utilizing proximity-based technology.

Key Feautures:
  • By automatically detecting students' proximity through their device location, the system significantly reduced the time required for teachers to take attendance
  • The utilization of a remote MySQL database enabled secure storage of attendance records
  • The application features personalized dashboards catering to teachers, students, and administrators, also enabled database querying and management options for administrators
  • This innovative solution streamlined attendance management, improving accuracy and accountability while providing valuable insights through data visualization
  • The Web application used Selenium JS API for functional tests and K6 for load testing, ensuring a seamless experience to users
